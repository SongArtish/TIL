## Overview

cURL은 command line 기반의 **웹 요청 도구**이다. Unix, Linux, Windows 등의 주요 OS에서 구동이 가능하며, HTTP/HTTPS/FTP/LDAP/SCP/TELNET/SMTP/POP3 등 핵심 프로토콜을 지원한다. 다운로드 및 업로드가 모두 가능하다.



## 명령어

아래와 같은 형태로 사용된다.

```shell
curl [options] <url>
```

`curl www.google.com` 명령어를 입력하면 HTML 정보를 콘솔에 출력해준다.

```shell
curl <url>
```
