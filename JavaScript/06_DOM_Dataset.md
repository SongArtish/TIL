## Overview

`dataset`을 통해 태그 내에 선언한 데이터 속성을 가져올 수 있다.

```html
<!-- html-->
<div data-user="SongArtish" data-role="moderator" data-user-id="bulgen">
    Song Artish
</div>
```

```javascript
// JavaScript
$0.dataset.user // 'SongArtish'
$0.dataset.role // 'moderator'
$0.dataset.userId // 'bulgen'
```
