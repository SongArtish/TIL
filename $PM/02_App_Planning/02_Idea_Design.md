## Persona

> 특정한 사용자의 가상 버전

- 기획, UX기획, 마케팅에까지 필요한 도구



### 구성 요소

| 구성 요소                      | 목적                                                         |
| ------------------------------ | ------------------------------------------------------------ |
| **고객명**                     | 커뮤니케이션을 위해                                          |
| **고객 정보**                  | 유입될 고객들의 특징(정량적)                                 |
| **고객의 태도**                | 해당 고객은 어떠한 성향을 가졌는가?(정성적)                  |
| **사용자의 니즈**              | 어떠한 이유로 들어왔는가?<br />어떠한 서비스 때문에 들어왔는가? |
| **이용에 대한 목표**           | 사용을 통한 최종 목표                                        |
| **서비스 프로세스별 이용행태** | 위 5가지를 통하여, 생각한 고객은 어떻게 이용할 것인가?<br />해당 기능을 왜 이용할 것인가? |

- 기획 + 마케팅을 할 때의 페르소나

| 구성 요소       | 목적                                      |
| --------------- | ----------------------------------------- |
| **락인 요소**   | 고객이 지속적으로 이용하는 요인           |
| **후킹 포인트** | 고객이 가장 매력적으로 느낄 포인트        |
| 그 외 기타      | (본인이 필요한 부분)<br />ex. 수익모델 등 |



### 고객 정보 분할

1. 본인이 생각하는 가장 가까운 타겟층부터 생각
   - 서비스에 맞게 적합한 척도로 구성한다.
   - ex) 나이, 성별, 직업, 주요 활동 지역, 외식 빈도, 카페 이용 빈도, 프랜차이즈 이용 척도
2. 해당 카테고리부터 분류 시작

- 타겟층이 세부적일 수록 효과 UP!



### Summary

1. **고객 정보** 분할부터 명확히
2. 주요 타겟층을 선정했다면, 페르소나를 통해 확인해보기
3. 페르소나를 지속적으로 진행해보면서 기능과 명확한 타겟층 찾기
4. 최종적으로 선정된 타겟층에게 '가볍게' 검증해보기
5. 서비스 기능, 타겟층 정리!
